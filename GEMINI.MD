# Developer Profile & Project Context

## Tech Stack & Architecture
- Primary frontend stack: React, TypeScript, and Tailwind CSS.
- Primary backend stack: FastAPI and Python.
- Always use strict typing in TypeScript; avoid the use of `any`.
- Keep React components modular and leverage functional programming paradigms.

## Communication & Explanation Rules
- Before writing extensive code, briefly outline your architectural approach.
- **Mathematical Rigor:** When queried about mathematical concepts, provide rigorous, pure mathematical explanations. Do not simplify or dilute math concepts with computer science-oriented examples, metaphors, or analogies. Treat mathematical queries as if speaking directly to a pure mathematics researcher.
